# 13 — Catalog: `script.*` + `signal.*` (Phase 3 work-unit #3)

> The `script.*` category turns TerraVolt into a **real coding partner**: the agent can read/write
> `.gd` and `.cs` files, compile-check them, search them, and refactor them. The `signal.*` category
> complements `node.*` by adding signal-graph–oriented operations (declare, list, bulk wire).

## 13.6 `script.*` — 8 tools

### `script.list`

- **Purpose:** enumerate every script file in the project.
- **Inputs:**
  `{ pattern?: string (glob, default "**/*.{gd,cs,vshader,shader}"), include_addon?: bool (default false) }`.
- **Outputs:**
  `{ scripts: [{ path, language: "gd"|"cs"|"vshader"|"shader", size_bytes, modified_at, is_tool: bool, has_class_name: bool, class_name?: string }], total }`.
- **Godot APIs:** `EditorFileSystem.get_filesystem()` walk; quick AST sniff for `@tool` and
  `class_name`.
- **safe:** true. **mutates:** false.
- **Cursor prompt:** _"List every gd script in res://scripts/enemies."_

### `script.read`

- **Purpose:** read a script's contents (envelope-aware).
- **Inputs:**
  `{ path: ResourcePath, range?: { start_line: int, end_line: int }, format?: "text"|"chunks" }`.
- **Outputs:**
  `{ path, language, size_bytes, content?: string, chunks?: [{ start_line, end_line, content }], truncated: bool }`.
- **Godot APIs:** `FileAccess.open(path, READ)` for plain text; for `Script` resources inside
  `.tscn` use `Script.source_code`.
- **safe:** true. **mutates:** false.
- **Errors:** `script.path_not_found` (`-33600`).
- **Cursor prompt:** _"Read player.gd lines 1-120."_

### `script.write`

- **Purpose:** create or overwrite a script file with content.
- **Inputs:**
  `{ path: ResourcePath, content: string, mode?: "overwrite"|"create_only" (default "overwrite"), if_match?: opaque-revision }`.
- **Outputs:** `{ written: true, path, bytes_written, lines: int, revision }`.
- **Godot APIs:** `FileAccess.open(path, WRITE)`; trigger `EditorFileSystem.scan()` to reindex.
- **safe:** false. **mutates:** true.
- **Errors:** `script.path_exists` (`-33601`) when `mode=create_only`,
  `protocol.idempotency_conflict`.
- **Cursor prompt:** _"Write a new file res://scripts/Boss.gd with this content..."_

### `script.patch`

- **Purpose:** apply a structured edit (line-range replace, or `@@`-style hunks) without resending
  the whole file.
- **Inputs:**
  `{ path: ResourcePath, hunks: [{ start_line: int, end_line: int, replacement: string }], if_match?: revision }`.
- **Outputs:** `{ patched: true, hunks_applied: int, lines_before, lines_after, revision }`.
- **Godot APIs:** same as `script.write`; pre-flight hunks against current text.
- **safe:** false. **mutates:** true.
- **Errors:** `script.patch_conflict` (`-33602`).
- **Cursor prompt:** _"Replace lines 40-55 of player.gd with this implementation..."_

### `script.validate`

- **Purpose:** compile-check a script.
- **Inputs:** `{ path: ResourcePath, mode?: "syntax"|"deep" (default "syntax") }`.
- **Outputs:**
  `{ ok: bool, errors: [{ line, col, message, severity }], warnings: [...], duration_ms }`.
- **Godot APIs:** `GDScript.new(); script.source_code = ...; script.reload()` returns `Error`; for
  headless: spawn `godot --headless --check-only --script <path>`. `.cs`: invoke `dotnet build` via
  Godot's MSBuild integration (`godot --headless --build-solutions`).
- **safe:** true. **mutates:** false.
- **Errors:** `script.dotnet_unavailable` (`-33603`), `script.validate_timeout` (`-33604`).
- **Cursor prompt:** _"Compile-check player.gd."_

### `script.find_usages`

- **Purpose:** locate references to a class, function, signal, or constant across `.gd` / `.cs` /
  `.tscn` / `.tres`.
- **Inputs:**
  `{ symbol: string, kind?: "class_name"|"function"|"signal"|"constant"|"any" (default "any"), case_sensitive?: bool (default true) }`.
- **Outputs:**
  `{ usages: [{ path, line, col, snippet, kind, confidence: "exact"|"heuristic" }], truncated: bool }`.
- **Godot APIs:** custom indexer; ripgrep-style scan across `res://`; cross-reference with
  `Script.get_method_list()`/`get_signal_list()` for declared sites.
- **safe:** true. **mutates:** false.
- **Cursor prompt:** _"Where is the `take_damage` method called?"_

### `script.rename_symbol`

- **Purpose:** rename a function / signal / class member project-wide (best-effort, with dry-run).
- **Inputs:**
  `{ scope: ResourcePath | "project", from: string, to: string, kind: "function"|"signal"|"constant"|"variable"|"class_name", dry_run?: bool, exclude?: [glob] }`.
- **Outputs:**
  `{ edits: [{ path, line, col, before, after }], applied: bool, dry_run: bool, files_changed: int }`.
- **Godot APIs:** uses `script.find_usages` results then `script.patch`; for `class_name` renames
  also rewrites scene-side type references where safe.
- **safe:** false. **mutates:** true.
- **Errors:** `script.rename_conflict` (`-33605`) if target name already exists in scope.
- **Cursor prompt:** _"Rename function `move_player` to `move_actor` everywhere."_

### `script.format`

- **Purpose:** format a script per the project's GDScript style guide.
- **Inputs:** `{ path: ResourcePath, in_place?: bool (default true) }`.
- **Outputs:** `{ formatted: bool, path, lines_before, lines_after, diff_summary }`.
- **Godot APIs:** delegate to `gdformat` if installed (`packages/godot-mcp-addon/tools/format.gd`
  calls `OS.execute("gdformat", [...])`); fall back to a built-in minimal pretty-printer (consistent
  tabs, surrounding-line discipline). `.cs`: delegate to `dotnet format`.
- **safe:** false. **mutates:** true.
- **Errors:** `script.formatter_missing` (`-33606`).
- **Cursor prompt:** _"Format player.gd."_

## 13.7 `signal.*` — 10 tools

> Signals = the heart of Godot's event model. These tools complement `node.modify`'s ad-hoc connect
> ops with a category dedicated to graph-shaped operations (read the graph, find dangling sinks,
> wire many sources at once).

### `signal.list_declared`

- **Purpose:** list user-declared signals on a node's script (not engine-inherited).
- **Inputs:** `{ path: NodePath }`.
- **Outputs:** `{ declared: [{ name, args: [{ name, type }], source_line }] }`.
- **Godot APIs:** parse script `signal` declarations with the GDScript tokenizer; cross-check with
  `Script.get_signal_list()`.
- **safe:** true. **mutates:** false.
- **Cursor prompt:** _"Which signals does /root/Main/Player declare?"_

### `signal.add_declaration`

- **Purpose:** add a `signal foo(arg: Type, ...)` declaration to a script.
- **Inputs:**
  `{ script_path: ResourcePath, signal_name: string, args?: [{ name, type }], doc_comment?: string }`.
- **Outputs:** `{ added: true, line, revision }`.
- **Godot APIs:** locate insertion point (after existing signal block or after
  `extends`/`class_name` lines) and `script.patch`.
- **safe:** false. **mutates:** true.
- **Errors:** `script.path_not_found`, `signal.name_exists` (`-33700`).
- **Cursor prompt:** _"Add signal `boss_died(boss_id: String)` to res://scripts/Boss.gd."_

### `signal.remove_declaration`

- **Purpose:** remove a `signal` declaration from a script.
- **Inputs:** `{ script_path: ResourcePath, signal_name: string }`.
- **Outputs:** `{ removed: true, line, revision }`.
- **Errors:** `signal.unknown` (`-33701`).
- **Cursor prompt:** _"Remove the `unused_event` signal from Boss.gd."_

### `signal.connect`

- **Purpose:** connect a signal (convenience over `node.modify { kind:'connect' }`).
- **Inputs:**
  `{ from_path: NodePath, signal_name: string, to_path: NodePath, method: string, flags?: int, binds?: any[] }`.
- **Outputs:** `{ connected: true, from_path, signal_name, to_path, method, state }`.
- **Godot APIs:** `Object.connect(signal, Callable(target, method).bindv(binds), flags)`.
- **safe:** false. **mutates:** true.
- **Errors:** `signal.unknown`, `signal.target_unknown` (`-33702`), `signal.method_unknown`
  (`-33703`).
- **Cursor prompt:** _"Connect Player.health_changed to HUD.update_health."_

### `signal.disconnect`

- **Purpose:** drop a single connection.
- **Inputs:** `{ from_path: NodePath, signal_name: string, to_path: NodePath, method: string }`.
- **Outputs:** `{ disconnected: bool }`.
- **Godot APIs:** `Object.disconnect`.
- **safe:** false. **mutates:** true.
- **Cursor prompt:** _"Disconnect Player.died from HUD.show_game_over."_

### `signal.list_connections`

- **Purpose:** list all outgoing connections for a signal (or all signals) on a node.
- **Inputs:** `{ path: NodePath, signal_name?: string }`.
- **Outputs:** `{ connections: [{ signal, target_path, method, flags, binds }] }`.
- **Godot APIs:** `Object.get_signal_connection_list(signal)`.
- **safe:** true. **mutates:** false.
- **Cursor prompt:** _"Who's listening to /root/Main/Boss.died?"_

### `signal.find_listeners`

- **Purpose:** reverse lookup — given an emitter signal, find every connected listener method
  anywhere in the active scene.
- **Inputs:**
  `{ from_path: NodePath, signal_name: string, scope?: "scene"|"project" (default "scene") }`.
- **Outputs:** `{ listeners: [{ to_path, method, defined_in: ResourcePath, line? }] }`.
- **Godot APIs:** walk scene tree; `Object.get_signal_connection_list`; for `project` scope, scan
  persisted `.tscn`/`.tres` for `[connection]` sections.
- **safe:** true. **mutates:** false.
- **Cursor prompt:** _"Show every listener of the Boss.died signal across the project."_

### `signal.bulk_connect`

- **Purpose:** wire many source/target pairs in one atomic batch.
- **Inputs:**
  `{ connections: [{ from_path, signal_name, to_path, method, flags?, binds? }], if_match?: revision }`.
- **Outputs:** `{ applied: [...], skipped: [...] (with reason), revision }`.
- **Godot APIs:** transactional `Object.connect` loop inside one UndoRedo action.
- **safe:** false. **mutates:** true.
- **Cursor prompt:** _"Wire all enemies' died signal to the Score.tick_kill method."_

### `signal.bulk_disconnect`

- **Purpose:** disconnect many in one batch (mirror of `bulk_connect`).
- **Inputs:** `{ connections: [{ from_path, signal_name, to_path, method }] }`.
- **Outputs:** `{ disconnected: int }`.
- **safe:** false. **mutates:** true.
- **Cursor prompt:** _"Disconnect every UI button from MainMenu.handle_click."_

### `signal.graph`

- **Purpose:** export the signal graph (for documentation or analysis).
- **Inputs:**
  `{ scope?: "scene"|"project" (default "scene"), format?: "json"|"mermaid"|"dot" (default "json"), include_engine_signals?: bool (default false) }`.
- **Outputs:** `{ format, content_string }` or
  `{ format: "json", graph: { nodes: [...], edges: [...] } }`.
- **Godot APIs:** walk tree; collect signals + connections; for `project` scope, parse `.tscn` files
  for `[connection]` headers.
- **safe:** true. **mutates:** false.
- **Cursor prompt:** _"Export the current scene's signal graph as Mermaid."_

## 13.12 Risks & mitigations

| Risk                                                                      | Mitigation                                                                                |
| ------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------- | ------------------------------------------------- |
| `script.rename_symbol` rewrites the wrong identifier in comments/strings. | Token-level rewrites using the GDScript tokenizer; never use plain regex on `.gd` bodies. |
| `.cs` build is slow (cold dotnet restore).                                | Cache `bin/`/`obj/` between runs; surface progress event.                                 |
| `script.find_usages` produces false positives across languages.           | Tag each match with `confidence: "exact"                                                  | "heuristic"`and require`exact`for`rename_symbol`. |
| `signal.find_listeners` against the whole project is expensive.           | Throttle to `scope=scene` by default; require explicit opt-in for project-wide scan.      |
| `script.format` formatters not installed on CI runner.                    | `script.formatter_missing` autoHeal includes the exact install command.                   |
